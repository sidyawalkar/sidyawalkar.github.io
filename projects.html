Innovation

Here are some of the projects I have worked on:

  • Create a stock trading bot
  • Create a ray tracer
  • Create a chess engine

Create a stock trading bot

I used Python to create a stock trading bot. This bot included features likes scanning highs and lows over a period of time. The bot would then commit a trade on a market testing site while recording trading data on a output spreadsheet

Create a ray tracer

I used C# to create a ray tracer which created images. I really enjoyed working with 3d modeling. One of the challenges faced was the render time for each image.To Fix this, I made sure the ray tracer was parallelized with threads to improve the render time.

Create a chess engine

Creating a chess engine was my most favorite project. I have always loved chess, and enjoyed coding the functions for each pieces' movement. I utlized python to code the legal movement of each piece. I also created a GUI which displayed the chess moves in real time.
